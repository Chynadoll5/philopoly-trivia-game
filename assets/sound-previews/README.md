# Philopoly sound audition sources

These files are permanent project copies used by `sound-audition.html`. They are not connected to the live game's sound settings until the owner selects the winners.

All source clips come from [Mixkit Free Sound Effects](https://mixkit.co/free-sound-effects/) and are used under the [Mixkit Sound Effects Free License](https://mixkit.co/license/#sfxFree).
